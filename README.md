# browser
Browser es una plataforma web independiente para búsqueda, correo, privacidad y marketing digital. El MVP incluirá usuarios, roles, seguridad, panel admin, base de datos MySQL, correo interno, buscador inicial, clientes, campañas y leads, con PHP, PDO, Docker y arquitectura preparada para crecer en AWS.
